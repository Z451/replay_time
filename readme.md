# Code and Source Data for "Temporal manifold in human hippocampal activity"

This repository contains the Python scripts and preprocessed data used to generate all figures in the manuscript:

> Zhu et al., *Replay of factorized temporal journey*, bioRxiv (2025).  
> [https://doi.org/10.21203/rs.3.rs-6112006/v1](https://doi.org/10.21203/rs.3.rs-6112006/v1)

### Environment
The analyses were run using **Python 3.10.18**.  
All package versions are listed in `requirements.txt`.  
To replicate the environment:
```bash
pip install -r requirements.txt
